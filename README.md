# CourtSight

CourtSight is an AI-powered court intelligence platform designed to transform Indian legal documents into structured, searchable, and analyzable data.

## Planned Features

- Metadata Extraction
- Semantic Search
- Similar Case Retrieval
- Legal Case Clustering
- Anomaly Detection
- Court Analytics
- Reinforcement Learning-based Court Scheduling

## Dataset

OpenNyaya – Indian Supreme Court Judgments (1950–2023)

33,000+ OCR-cleaned judgments.
